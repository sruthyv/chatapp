# Sapaad.Online V3

Sapaad.Online V3 is a rails react application

## Getting Started

1. Clone the repo

   ```
   $ git clone git@github.com:sruthyv/chatapp.git
   $ cd chatapp
   ```

2. Install dependencies

   ```
   $ bundle install
   ```

3. Create and setup the database

   ```
   $ rails db:create
   $ rails db:migrate

   ```

4. Run application locally

   ```
   $ rails s

   ```
